# Portfolio Site

Personal portfolio hosted on GitHub Pages.

## Deploy to GitHub Pages

1. Push this folder to the repo:
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio"
   git remote add origin https://github.com/connorsensenig/Porfolio.git
   git push -u origin main
   ```

2. Go to the repo on GitHub → **Settings** → **Pages** → set Source to `main` branch, `/ (root)`.

3. Your site will be live at `https://connorsensenig.github.io/Portfolio` within a minute or two.
